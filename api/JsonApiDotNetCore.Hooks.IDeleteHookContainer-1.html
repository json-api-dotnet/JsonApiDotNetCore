BeforeDelete(IEntityHashSet<TResource>, ResourcePipeline)

Implement this hook to run custom logic in the DefaultResourceService<TResource> layer just before deleting entities of type TResource.

For the Delete pipeline, entities will typically contain one entity.

The returned may be a subset of entities, in which case the operation of the pipeline will not be executed for the omitted entities.

If by the deletion of these entities any other entities are affected implicitly by the removal of their relationships (eg in the case of an one-to-one relationship), the BeforeImplicitUpdateRelationship(IRelationshipsDictionary<TResource>, ResourcePipeline) hook is fired for these entities.
