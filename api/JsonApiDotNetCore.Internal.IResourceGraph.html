GetRelationship<TParent>(TParent, String)

Gets the value of the navigation property, defined by the relationshipName, on the provided instance.

Parameters

The navigation property name.
Returns
Type Description
System.Object
Type Parameters
Name Description
TParent
Remarks

In the case of a HasManyThrough relationship, it will not traverse the relationship and will instead return the value of the shadow property (e.g. Articles.Tags). If you want to traverse the relationship, you should use GetRelationshipValue<TParent>(TParent, RelationshipAttribute).

Examples
_graph.GetRelationship(todoItem, nameof(TodoItem.Owner));

GetRelationshipValue<TParent>(TParent, RelationshipAttribute)

Gets the value of the navigation property (defined by the RelationshipAttribute) on the provided instance. In the case of HasManyThrough relationships, it will traverse the through entity and return the value of the relationship on the other side of a join entity (e.g. Articles.ArticleTags.Tag).
