Class JsonApiQueryController<TResource, TId>

The base class to derive resource-specific read-only controllers from. This class delegates all work to BaseJsonApiController<TResource, TId> but adds attributes for routing templates. If you want to provide routing templates yourself, you should derive from BaseJsonApiController directly.
